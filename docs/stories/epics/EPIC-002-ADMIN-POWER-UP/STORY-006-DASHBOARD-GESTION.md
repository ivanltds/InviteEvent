# STORY-006: Dashboard de Gestão Inicial

## Descrição
Implementação da interface administrativa principal que serve como ponto de entrada para o organizador gerenciar o evento.

## Critérios de Aceitação
1. **Acesso Restrito:** Proteção por senha via variável de ambiente `NEXT_PUBLIC_ADMIN_PASSWORD`.
2. **Navegação Centralizada:** Links diretos para Gestão de Convidados, Lista de Presentes e Configurações.
3. **Layout Responsivo:** Sidebar ou Grid de cards acessível em dispositivos móveis e desktop.
