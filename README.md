# InviteEventAI - Wedding Management Platform 💍

InviteEventAI is a sophisticated wedding management platform built with a "CLI-First" philosophy, designed for high observability and seamless user experience. It handles everything from guest invitations and RSVP management to gift lists and visual storytelling.

## 🏛️ Architecture & Principles

This project follows the **AIOX (AI-Optimized eXperience)** framework, prioritizing:
1.  **CLI First:** Core logic is accessible and testable via CLI before the UI.
2.  **Observability Second:** Deep monitoring and logging across all layers.
3.  **UI Third:** A polished, responsive interface built on a solid foundation.

### Tech Stack
-   **Frontend:** Next.js 16 (React 19) with TypeScript
-   **Backend:** Next.js Server Actions & API Routes
-   **Database:** Supabase (PostgreSQL)
-   **Storage/Media:** Cloudinary
-   **Hosting:** Vercel
-   **Testing:** Jest 30 & React Testing Library
-   **Infrastructure:** AIOX-Core (Agent-driven development)

## 🛠️ Key Features

-   **RSVP System:** Real-time guest confirmation and attendance tracking.
-   **Gift List:** Integrated list with stock management and PIX payment support.
-   **Admin Dashboard:** Full control over event configuration, guest lists, and narratives.
-   **Visual Branding:** Fluid typography, custom themes, and hero image management via Cloudinary.
-   **Security Hardening:** Server-side auth, Supabase RLS, and obfuscated invite slugs.
