# STORY-045: Galeria de Fotos com Gestão de Álbuns

## Descrição
Como um casal, desejo subir fotos e organizá-las em álbuns (ex: "Ensaio Pré-Wedding", "Nossa História"), para compartilhar os momentos mais marcantes com os nossos convidados de forma elegante.

## Critérios de Aceitação
1. **CRUD de Álbuns:** Interface admin para criar, editar nome e deletar álbuns por evento.
2. **Masonry Grid:** Exibição pública das fotos usando layout tipo "Pinterest" (Masonry) para fotos de diferentes orientações.
3. **Lightbox Viewer:** Ao clicar na foto, abrir em tela cheia com navegação lateral (slideshow).
4. **Otimização Cloudinary:** Usar transformações automáticas (format e quality auto) para garantir performance no mobile.
