# STORY-015-RSVP-EXPORT: Exportação de Convidados (CSV)

## Descrição
Botão no Admin para baixar a lista completa de convidados e seus RSVPs em formato CSV.

## Critérios de Aceitação
1. Botão "Exportar CSV" na Gestão de Convidados.
2. Arquivo gerado com Nome, Tipo, Limite e Status do RSVP.
3. Suporte a caracteres especiais (UTF-8).
