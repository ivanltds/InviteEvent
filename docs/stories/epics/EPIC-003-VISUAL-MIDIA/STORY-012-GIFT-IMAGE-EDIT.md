# STORY-012: Edição Manual de Fotos de Presentes

## Descrição
Capacidade do organizador de alterar manualmente a imagem de um presente através do painel admin, enviando um novo arquivo via Cloudinary.

## Critérios de Aceitação
1. **Botão de Troca:** Opção de "Trocar Foto" na edição do presente.
2. **Integração Cloudinary:** Reaproveitamento da infra de upload do Widget.
3. **Persistência:** Atualização da `imagem_url` no banco de dados.
