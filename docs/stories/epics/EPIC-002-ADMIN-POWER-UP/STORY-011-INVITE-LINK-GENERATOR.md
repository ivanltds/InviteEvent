# STORY-011: Geração de Links Únicos

## Descrição
Capacidade de gerar e copiar links personalizados para cada convite, permitindo que o organizador envie acessos diretos via WhatsApp ou outros canais.

## Critérios de Aceitação
1. **Slug Automático:** Geração de slug baseado no nome principal do convite.
2. **Cópia para Clipboard:** Botão "Copiar Link" na tabela de convidados.
3. **Formato do Link:** URL no formato `BASE_URL/?invite=slug`.
